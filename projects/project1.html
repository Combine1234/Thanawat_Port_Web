Bottle Donation Machine with Recycling System

วัตถุประสงค์

โครงงานเครื่องบริจาคขวดน้ำพร้อมระบบแปรรูป เป็นโครงงานที่จะช่วยในการแก้ปัญหาด้านการจัดการยังให้มีประสิทธิภาพมากขึ้น โครงงานนี้สามารถรับขวดน้ำพลาสติก หรือกระป๋องโลหะที่ใช้แล้วและทำการบีบอัดเพื่อให้ขวดอยู่ในสภาพที่สามารถนำไปรีไซเคิลได้ง่าย ซึ่งช่วยลดปริมาณขยะพลาสติกที่ต้องจัดการและสร้างมูลค่าจากขยะที่แปรรูป นอกจากนี้ยังส่งเสริมการรีไซเคิลแล้วเครื่องบริจาคขวดน้ำพร้อมระบบแปรรูปยังมีระบบเก็บข้อมูลผู้ใช้ จำนวนการบริจาคของผู้ใช้คนนั้นๆ ลงบน Google Sheet และแสดงผมในรูปเว็ปไซต์ที่เราได้สร้างขึ้นมา นอกจากนี้ยังสามารถดูปริมาณขยะทั้งหมดที่เก็บอยู่ในถังบน Arduino Cloud

กระบวนการทำงาน

ระบบเครื่องบริจาคขวดน้ำถูกพัฒนาขึ้นเพื่อส่งเสริมการรีไซเคิลและลดปริมาณขยะ โดยมีขั้นตอนการทำงานที่ครอบคลุม เริ่มจากการรับขวดผ่านช่องรับที่สามารถตรวจสอบประเภทวัสดุของขวดได้ หลังจากนั้นผู้ใช้จะทำการยืนยันตัวตนด้วยการกรอกเลขประจำตัวนักเรียนหรือสแกนบัตร เพื่อสะสมแต้มจากการบริจาค จากนั้นขวดจะถูกส่งเข้าสู่ระบบบีบอัดเพื่อลดขนาดและประหยัดพื้นที่จัดเก็บ โดยขวดที่ถูกบีบอัดแล้วจะถูกแยกเก็บในช่องสำหรับพลาสติกและโลหะอย่างเป็นระเบียบ นอกจากนี้ ระบบยังเชื่อมต่อกับ IoT ผ่าน Arduino Cloud ทำให้สามารถติดตามและจัดการข้อมูลการบริจาคได้อย่างมีประสิทธิภาพ ทั้งนี้ การสะสมแต้มยังช่วยกระตุ้นให้ผู้คนมีส่วนร่วมในการรักษาสิ่งแวดล้อมและการรีไซเคิลมากยิ่งขึ้น
